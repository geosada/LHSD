# LHSD: Local Hessian Spectral Dimension
### This is the code used in the paper "[Local Hessian Spectral Filtering for Robust Intrinsic Dimension Estimation](https://arxiv.org/abs/2605.01221)" (ICML 2026).

This codebase is built entirely on Kamkari’s [FLIPD](https://arxiv.org/abs/2406.03537) codebase. 
We are grateful that their code enabled us to conduct our experiments smoothly. 
Almost all of the code used in our experiments is located under ``notebooks`` directory.
